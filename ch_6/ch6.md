## while语句
```
while(expression) statement
```
当expression的运算值不为0时则满足循环条件


这里的expression可以是赋值语句,比如a=1,while循环中每次进行条件判断语句时,都会先将a赋值为1,在得到a的值作为是否进入循环的条件,无限循环!其他语言这种写法可能编译不会通过,但是c中却可以,想象一下,你将a==1不小心写成了a=1,会是什么后果!

很多时候,可以将expression的常量放在运算符号的左边,即使你不小心少写成1=a,编译器也会帮你找出错误,常量是不允许赋值的

## 关系运算符

while循环中expression的结果往往是进行比较后的结果,这些运算需要关系运算符来组织

- \> 大于
- < 小于
- == 等于
- \> = 大于等于
- <= 小于等于
- != 不等于

字符串不能直接使用关系运算符,字符串底层是char数组,数组显然不能够进行关系运算.

## 优先级

关系运算符的优先级低于算术运算符

关系运算符之间的优先级
- 高优先级组 : \< <= > >=
- 低优先级组 : == !=

## for循环
```
for (initialize;test;update)statement
```
- initialize表达式是初始化,只有第一次进入循环会执行,一般初始化一个变量
- test表达式进行判断是否进入循环,一般用关系运算对初始变量进行运算
- update表达式进行更新,一般更新初始化一个变量的值

# 逗号表达式

逗号表达式（comma expression）在 C 语言中用于将多个表达式组合成一个表达式，并按顺序依次求值。它的形式为 expr1, expr2, expr3, ...，其中每个表达式都可以是任意合法的表达式。

逗号表达式的求值过程从左到右进行，每个表达式按照顺序依次求值。整个表达式的值是最后一个表达式的值。

过度复杂或滥用逗号表达式可能会导致代码难以理解和维护
```
int a=1,b=2; //定义多个变量
a=10,b=20; //赋值多变量
```
定义多个变量只支持同类型

应该没人会这么用:
```
h=100,200; //相当于h=100;200; 也就是说h=100
h=(100,200); //h=200;

h=(y=10,100+y/2+y*10); // 先给y赋值为10,在计算100+y/2+y*10,最后h=第二个表示结果
```

函数定义多个形式参数的时候也可以使用逗号表达式将各个参数隔开

在for循环里面可以通过逗号表达式初始化多个参数,更新多参数.

## do while 循环
```
do 
    statement 
while(expression);
```
`do while` 是一条语句,所以要以;结尾,可以理解为就是语法规定以分号结尾.

## 数组

定义数组变量,变量记录了该数组的起始地址,寻址的时候按照规定的数据类型大小进行寻址.

c的数组并不会检查角标越界问题,越界可能会带来这些问题:
- 访问越界：访问到未分配给数组的内存空间。这可能导致程序崩溃、数据损坏或未定义的行为。
- 内存安全问题：可能会影响到其他变量或数据结构的内存区域，导致内存安全问题，如缓冲溢出、数据覆盖等。
