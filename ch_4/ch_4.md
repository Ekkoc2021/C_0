# ch4

## 字符串
字符串常量:"hello world!"==等价字符数组==>"hello world!\0"
处理字符串的函数底层按照字符数组处理字符串,以\0表示字符串的结束.

字符串常量赋值可以赋给字符数组,这个字符数组的长度至少要大于这个字符串长度+1.

如果赋值的数组的长度小于最小长度,会出现缓冲区溢出.

多行字符串常量:
两个连续的字符串之间只是空白的,只要中间是空白的c会将他们连接为一个字符串.
```
char str1[14]="hello world!";
char str2[14]="hello "
"world!";
```

**c语言中字符串不支持加减运算.**

scanf函数,使用%s说明符读取输入时会将空格作为一个字符串结束标志,将输入的作为一个字符常量赋值给对应的字符数组.

sizeof 运算符号,计算类型变量占了多少字节, 当是类型时必须在类型上加上括号就像使用函数那样,但是实际并不是函数,变量也可以用括号也可以用空格与sizeof隔离开.

strlen函数,用于计算字符串的长度(\0标志着字符串的结束)

## 常量 

```
#define PI 3.14
```
命名满足变量命名的规则,通常常量为大写,编译时会将3.14替代所有PI出现的位置.

```
const int PI=3.14
```
用const关键字修饰变量,表示该变量为常量

float.h和limits.h两个头文件中定义了一些系统常量,需要时可以声明并使用,如float.h中的FLT_MAX,limits.h中的INT_MAX

# scanf

如果是读取数字,他会读取输入串中的第一个数字,输入的开始部分非数字部分(遇到字符则就不读读取了不会赋值任何数值给变量,如果是空格,回车会被丢弃)会被丢弃.

输入的数据会被缓存下来,如果本次读取没有读取完所有数据剩余的会被下次读到.

读取字符串会以字符开始以空白作为一个字符串的结束

格式化输入:
```
scanf("%d,%d",&a,&b); 
//意味着会读取两个整数,整数之间用逗号分割如输入:1,2
//这样输入也是合法的: 1,     2
```

返回值,没有读取到任何项目,返回0,读取到返回1

# *在printf和scanf中作用
```
prinf("%*d",width,num);
//width会取代*位置表示输出整数的格式,num真正输出数字
scanf("%*d %*d %d",&num); 
//录入的前两个数字会被丢弃,只有最后一个成功复制,读取特定的列会非常有用
```
